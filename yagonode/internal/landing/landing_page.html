This is a self-hosted YagoSeek search node — a YaCy-compatible peer built on the yago toolkit. It stores and serves a Reverse Word Index (RWI), participates in the YaCy peer-to-peer network, and answers local and federated searches.

This node speaks the YaCy peer protocol here.

If you believe this node is misbehaving on the YaCy network, please report it so the operators can address it:

github.com/D4rk4/yago/issues
